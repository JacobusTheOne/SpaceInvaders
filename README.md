# Space Invaders

A Space Invaders remake built with Unreal Engine 5.7.

## Requirements

- Unreal Engine 5.7
- Visual Studio 2022 (with C++ game development workload)

## Getting Started

1. Clone the repository
2. Right-click `SpaceInvaders.uproject` → **Generate Visual Studio project files**
3. Open `SpaceInvaders.sln` in Visual Studio and build (Development Editor)
4. Open `SpaceInvaders.uproject` in Unreal Engine 5.7

## Project Structure

- `Source/SpaceInvaders/` — C++ game module
- `Config/` — Project configuration files
- `Content/` — Game assets (blueprints, meshes, materials, levels)
